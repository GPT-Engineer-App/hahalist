# hahalist

Develop a simple and intuitive To-Do Task manager app named "Ha-ha List". Implement basic features for task management including creation, editing, and deletion of tasks. Ensure tasks can be categorized (e.g., personal, work, shopping) and have options to set due dates and reminders.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/hahalist.git
cd hahalist
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
